# AxylOS 2024.1 Release: What's New

Discover the latest features, improvements, and optimizations in our newest release including enhanced performance and new desktop themes.

---

## Highlights

- **Enhanced Performance:** Major optimizations across the desktop and core system for a smoother experience.
- **New Desktop Themes:** Fresh, modern themes for a beautiful and customizable look.
- **Updated Software:** The latest versions of your favorite applications and tools.
- **Improved Installer:** Easier and faster installation process with better hardware detection.
- **Community Contributions:** Thanks to everyone who contributed bug reports, code, and feedback!

---

## Detailed Changes

### Performance Improvements

AxylOS 2024.1 brings significant performance boosts, especially on lower-end hardware. Boot times are faster, and memory usage has been reduced across the board.

### New Themes

We've added several new desktop themes, including both light and dark variants. You can switch themes easily from the settings menu.

### Updated Applications

All core applications have been updated to their latest stable versions, including the terminal, file manager, and browser.

### Installer Enhancements

The installer now supports more hardware out of the box and provides clearer instructions for new users.
