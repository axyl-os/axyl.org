## Installation Guide

1. **Download the ISO**  
   Visit the [AxylOS Download Page](https://axyl.org/download) and download the latest ISO.

2. **Create a Bootable USB**  
   Use a tool like [balenaEtcher](https://www.balena.io/etcher/) or `dd` to write the ISO to a USB stick.

3. **Boot from USB**  
   Insert the USB stick into your computer and boot from it.

4. **Follow the Installer**  
   The AxylOS installer will guide you through the installation process.

## First Steps

After installation, here are some essential configuration steps:

- Set up your user account and password.
- Connect to Wi-Fi or Ethernet.
- Update your system:  
  ```sh
  sudo pacman -Syu
  ```
- Explore the pre-installed applications.
