this is the source code for the [AxylOS Website](https://axyl.org) there are a few things about this site that you should be aware of

it uses Motion UI and Shadcn UI as well as Tailwind CSS, nextjs, and react for the visual design and functionality of the website. It uses Better Auth for the Oath and Login Functionalities and Stripe for the payment and donation functionalities.

We plan on migrating to the better auth infrastructure in the future. This infra is still in a closed beta at this time though so we have delayed moving to that for the time being though out code is being written to account for the planned migration.

the community is located on discord [here](https://awfixer.link/discord)

the entirity of the AxylOS project is developed by [awfixer](https://awfixer.me) with the backing of people like [FOSS Torrent](https://fosstorrents.com) and [Open Status](https://openstatus.dev) we thank both of them for the parts they have provided to use to help start this project again.

We have decided to continue this project to help the community in honor of Arco Linux, we thank erik for everything that he provided the linux community during his time maintaining the project.
